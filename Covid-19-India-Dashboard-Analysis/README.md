## 📌 Project Overview

This project presents an **interactive Covid-19 Dashboard for India** built using **Tableau**. The dashboard analyzes India's Covid-19 data from multiple perspectives, including confirmed cases, recoveries, deaths, vaccination progress, testing statistics, age distribution, gender distribution, and state-wise analysis.

The dashboard combines multiple datasets into a single interactive visualization, enabling users to explore pandemic trends and gain meaningful insights through filters, maps, charts, and dashboards.

- Study vaccination progress
- Analyze testing statistics
- Compare age group distribution
- Compare gender distribution
- Display geographical insights using India Map
- Build an interactive Tableau Dashboard

# 📈 Key Insights

- Maharashtra recorded one of the highest Covid-19 death counts.
- The 20–39 age group contributed the largest share of reported cases.
- Covishield accounted for the majority of vaccine doses administered.
- First dose administration significantly exceeded second dose administration during the observed period.
- Testing volumes varied considerably across Indian states.
- The dashboard enables quick comparison of pandemic trends across regions.
