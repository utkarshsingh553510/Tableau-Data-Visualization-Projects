

# 💻 Installing Tableau

Tableau install karne ke liye official website par jao aur **Download Tableau** option select karo.

Installation ke time mainly **2 options** milte hain:

## 🎓 1. Tableau for Students

Ye option **students** ke liye hota hai.

Isme free access ke liye apni student details verify karni padti hain, jaise:

- 📧 College Email ID
- 🪪 Student ID Card / Student Proof
- 🏫 College/University Details

Verification complete hone ke baad student version use kar sakte ho.

> 💡 Agar tumhare paas valid college email ya student proof hai, to ye best option hai.

---

## 🌍 2. Tableau Public

Ye version **sabhi users** ke liye free hai.

Isme student verification ki zarurat nahi hoti.

Features:
- ✅ Completely Free
- ✅ Koi Student ID ki need nahi
- ✅ Learning aur Practice ke liye best
- ⚠️ Dashboards public hote hain (Private save nahi kar sakte)

> 💡 Beginners ke liye **Tableau Public** sabse achha option hai.

---

## 📌 Quick Comparison

| Tableau for Students | Tableau Public |
|----------------------|----------------|
| Student Verification Required | ❌ No Verification |
| Student ID/College Email Required | ❌ Not Required |
| Learning + Full Features | Learning & Practice |
| Private Work Supported | ❌ Public Saving Only |

> **💡 Recommendation:** Agar tum student ho aur valid college proof hai, to **Tableau for Students** use karo. Agar sirf practice karni hai, to **Tableau Public** best choice hai.
