# 📊 Joining Multiple Visualizations Using Dashboard

Jab hum Tableau me do ya do se zyada **Worksheets (Charts/Visualizations)** ko ek hi screen par dikhana chahte hain, tab hum **Dashboard** ka use karte hain.

> Dashboard ka matlab hai **multiple visualizations ko ek jagah combine karna**, taaki data ko ek saath analyze kiya ja sake.

---

## 🛠️ Steps

### Step 1

Sabse pehle apni alag-alag Worksheets bana lo.

Example:

- **Sheet 1** → Line Chart
- **Sheet 2** → Bar Chart

---

### Step 2

Neeche diye gaye **New Dashboard (📋)** icon par click karein.

👉 Isse ek naya Dashboard create ho jayega.

---

### Step 3

Left side me **Size** option me **Automatic** select karein.

Isse Dashboard screen size ke according automatically adjust ho jayega.

---

### Step 4

Left side ke **Sheets** section se **Sheet 1** aur **Sheet 2** ko Dashboard me drag & drop karein.

Ab dono visualizations ek hi Dashboard me dikhne lagenge.

---

## 📦 Arrangement Options

Dashboard me charts ko arrange karne ke liye Containers ka use hota hai.

### 🟦 Horizontal Container

Agar **Horizontal Container** use karte hain to charts **Left → Right** arrange hote hain.

Example:

```
+-------------+-------------+
|  Sheet 1    |   Sheet 2   |
+-------------+-------------+
```

---

### 🟩 Vertical Container

Agar **Vertical Container** use karte hain to charts **Top → Bottom** arrange hote hain.

Example:

```
+-------------+
|  Sheet 1    |
+-------------+
|  Sheet 2    |
+-------------+
```

---

## ✏️ Rename Worksheet

Default names jaise **Sheet 1** aur **Sheet 2** ko change bhi kar sakte hain.

### Steps

1. Sheet Tab par Right Click karein.
2. **Rename** select karein.
3. Naya naam de dein.

Example:

- Sheet 1 → Sales Trend
- Sheet 2 → Genre Analysis

---

## 💡 Real-Life Example

Ek Video Game Sales Dashboard me:

- 📊 Bar Chart → Genre-wise Global Sales
- 📈 Line Chart → Year-wise Global Sales

Dono charts ko Dashboard me combine karke ek hi screen par analysis kiya ja sakta hai.

---

## ✅ Summary

| Option | Purpose |
|---------|---------|
| New Dashboard | Multiple worksheets ko combine karna |
| Automatic Size | Dashboard ko screen ke according adjust karna |
| Horizontal Container | Charts ko side-by-side dikhana |
| Vertical Container | Charts ko upar-niche dikhana |
| Rename Sheet | Worksheet ka naam change karna |

> **Note:** Dashboard ka use multiple charts ko ek hi screen par combine karke ek interactive report banane ke liye kiya jata hai
