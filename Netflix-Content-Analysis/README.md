# 📖 Overview

This project presents an interactive Tableau dashboard for analyzing Netflix Movies and TV Shows. The dashboard enables users to explore Netflix's content library using dynamic filters and visualizations, helping uncover trends across genres, ratings, countries, release years, and content types.

# 📈 Key Insights

- 🎬 Movies represent approximately **68%** of Netflix's content library.
- 📺 TV Shows account for around **32%** of the available titles.
- ⭐ **TV-MA** is the most common content rating.
- 📅 Netflix experienced significant content growth after **2016**, peaking around **2019**.
- 🎭 Documentaries and Stand-Up Comedy are among the most common genres.
- 🌍 The United States contributes the highest number of Netflix titles.
