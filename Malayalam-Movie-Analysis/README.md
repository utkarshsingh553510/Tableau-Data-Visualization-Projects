# 📌 Project Overview

This Tableau dashboard provides a comprehensive analysis of the Malayalam film industry by exploring financial performance, return on investment (ROI), star power, award impact, and the effects of the COVID-19 pandemic on movie profitability.

The dashboard transforms raw movie data into actionable insights using interactive visualizations, enabling users to identify industry trends and evaluate factors that influence box office success.

---

# 🎯 Objectives

- Analyze movie profitability and ROI.
- Compare the performance of different star power tiers.
- Measure the impact of awards on financial success.
- Study the effect of COVID-19 on Malayalam cinema.
- Discover high-performing movies with efficient budgets.

# 📈 Key Insights

- Award-winning movies generally achieve higher average ROI than non-award-winning films.
- Post-COVID Malayalam cinema experienced a strong financial recovery.
- Budget alone does not guarantee commercial success.
- Several low-budget films generated exceptional returns.
- Higher star power often contributes to improved box office performance.
